# Abstract

The goal of this guide is to help Ripple users know what to do
when an error occurs.
The "Generic errors" section goes through known issues encountered
while learning Ripple, on any supported target architecture,
and their solutions.
The "Hexagon-specific errors enumerates known issues found when targeting
Hexagon, along with their solutions.
