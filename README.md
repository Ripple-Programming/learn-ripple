## Documentation source for Ripple.

This repository hosts the documentation for **Ripple**. The documentation is hosted
here: <https://github.qualcomm.com/pages/CompilerLabs/Ripple/>.

If you are looking for the mirrored source-code of our LLVM-fork, check the
[mirror](https://github.qualcomm.com/CompilerLabs/Ripple/tree/mirror) branch.

### HOWTO: Build the docs

1. `cargo install mdbook`
2. `mdbook build -d build`
